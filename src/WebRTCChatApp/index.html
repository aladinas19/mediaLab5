WebRTCChatApp (mediaLab5)

Serverless WebRTC per DataChannel. Kairėje – Create (offer), dešinėje – Join (answer).

Veikimo seka: 1) Create nukopijuok Offer → 2) Join įklijuok Offer ir Start → 3) Join nukopijuok Answer → 4) Create įklijuok Answer ir Start → 5) Create įjunk kamerą ir Start sending frames.
